1. Cài đặt Project:
    npx create-react-app ten-app
2. chạy Project
    2.1/ mở terminal ngay tại file package.json
    2.2/ gõ lệnh: npm run start
3. Tài liệu về reactJS:
4. flow code reactJS: index.html => index.js => app.js
5. reactJS sẽ không hiểu HTML, mà nó chỉ hiểu JSX (class => class.jsx)
6. lý do dùng component là: 
    - dễ quản lý (chia để trị)
    - tái sử dụng (ưu tiên hàng đầu).
7. Điều kiện để tạo ra một component:
    - tái sử dụng (ưu tiên hàng đầu)
    - dễ quản lý (chia để trị)
8. Có 2 dạng component:
    - class
    - function
9. Data-binding: là cách đưa biến hay thuộc tính js ra ngoài html
10. state: 
    - dùng để những dữ liệu của component
    - giúp đồng bộ UI và Value của component
11. props:
    - props là thuộc tính cảu component (property)
    - props dùng để truyền data từ component => component
12. điều kiện để lưu state:
    - là nơi chứa các nút xử lý
    - là nơi chứa các giao diện hiển thị
13. state định nghĩa component nào thì các hàm xử lý setState phải định nghĩa ở component đó
